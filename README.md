# FocusSharp ⏱️

> Focus time, your way.

FocusSharp is a minimal yet powerful focus timer and time tracking web app — built for people who want to know where their time actually goes, without the complexity of traditional Pomodoro apps.

No rigid 25/5 splits. No task management. No gamification. Just clean, honest focus tracking by category.

🌐 **Live:** [focussharp.app](https://focussharp.app)

---

## Why FocusSharp?

Most Pomodoro apps force you into fixed intervals and don't meaningfully track _what_ you're working on. FocusSharp is different:

- **You set the duration** — 25 min, 40 min, 90 min, whatever fits your flow
- **You decide the break** — timed (5/10/15 min) or open-ended ("I'll be back when I'm ready")
- **You see where time goes** — pie chart breakdown by category, daily/weekly/monthly

---

## Features

- ⏱️ **Flexible session timer** — slider + quick chips, circular progress ring, pause/resume
- 🏷️ **Custom categories** — name them, color them, track by them
- ☕ **Flexible break flow** — timed break or open break, you decide after every session
- 📊 **Stats dashboard** — donut chart, daily totals, category breakdown, period comparison
- 🌙 **Dark mode** — full support out of the box
- 📱 **Mobile first** — designed to feel like a native app
- 🔒 **No account required** — works offline, data stays on your device

## Getting Started

```bash
# Clone the repo
git clone https://github.com/yourusername/focussharp.git
cd focussharp

# Install dependencies
npm install

# Run locally
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
